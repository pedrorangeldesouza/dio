# 📘 IA Criando Conteúdo Realista

## 📒 Descrição
Este projeto foi criado para o desafio Natty or Not da DIO.

Utilizei inteligência artificial para gerar um mini e-book com prompts úteis para estudantes e programadores que desejam utilizar IA no aprendizado.

## 🤖 Tecnologias Utilizadas
- ChatGPT
- Canva
- GitHub

## 🧐 Processo de Criação
Primeiro utilizei o ChatGPT para gerar o conteúdo do e-book com prompts úteis de inteligência artificial.

Depois organizei o conteúdo no Canva para criar um e-book visual em PDF.

Por fim publiquei o material no GitHub como parte do desafio da DIO.

## 🚀 Resultados
Foi criado um mini guia contendo 10 prompts de IA que ajudam estudantes e programadores a estudar, gerar ideias e aprender tecnologia com mais eficiência.

## 💭 Reflexão
Criar conteúdo com IA é rápido e poderoso, mas ainda é importante revisar e organizar o material para que ele fique realmente útil e natural.
